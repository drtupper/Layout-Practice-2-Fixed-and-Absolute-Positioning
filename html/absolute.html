Absolute Position

An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed). However; if an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling. The top, right, bottom, and left properties are used to position the element.

Setting position to absolute removes an element from the flow, and it is ignored by the elements around it.
